# Codebase: Blacklight Web Designs

## Overview
This is the central workspace for **Blacklight Web Designs**, a top-tier professional web design business. It contains the business identity, branding guidelines, operational workflows, and the technical toolkit (Antigravity Kit) used to deliver high-performance digital experiences.

## File Dependencies
- `branding-guide.md` → All future UI components and designs.
- `onboarding-flow.md` → All project planning phases.
- `.agent/scripts/checklist.py` → All verification phases.

## Directory Structure
- `.agent/`: Specialist agents, skills, and master scripts.
- `branding-guide.md`: Core visual standards.
- `business-identity.md`: Mission and values.
- `lead-gen-strategy.md`: Inbound and outbound lead tactics.
- `marketing-plan.md`: Social media strategy and copy.
- `onboarding-flow.md`: Client engagement process.
- `portfolio-specs.md`: Standards for project showcases.
- `pricing-model.md`: SA-specific tiered pricing.
- `service-agreement-template.md`: Professional client contract.
- `service-offerings.md`: Tiered service definitions.
- `task.md`: Current operational task list.
